Dedicated to all Kraftwerk fans.

This project uses an Arduino Lilypad microcontroller board, a battery holder for CR 2032, 8 330 OHM resistors and 8 red 3mm diameter LEDs.
In this folder there are the wiring diagram of the connections, the BOM (Bill Of Materials) html file, the software to be loaded via the Arduino IDE and
the mp3 track "Music Non Stop".
